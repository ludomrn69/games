🐺 Loup-Garou

• Le village cache des loups-garous. La nuit, les loups dévorent un villageois ; la voyante sonde l'identité d'un joueur.

• Le jour, tout le monde vote pour éliminer un suspect. On révèle son rôle.

• Les villageois gagnent s'ils éliminent tous les loups ; les loups gagnent s'ils égalent les villageois.

• La voyante peut accuser publiquement pour rallier les votes !
